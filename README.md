# 🍰 Tiệm Bánh :)

Website bán bánh đơn giản được xây dựng bằng HTML, CSS và JavaScript.

## 🚀 Tính năng
- 🧁 Hiển thị 20 loại bánh
- 🛒 Thêm vào giỏ hàng
- 💳 Thanh toán
- ✔ Xác nhận mua hàng
- 📦 Cập nhật số lượng sản phẩm

## 📁 Công nghệ sử dụng
- HTML
- CSS
- JavaScript (LocalStorage)
